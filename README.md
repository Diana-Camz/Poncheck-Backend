# Poncheck Backend

## Description

Poncheck Backend is a REST API developed with Java and Spring Boot for managing products, categories, and sales in a local ponche business.

The goal of this project is to provide a backend system that helps organize product inventory, register sales, manage product categories, and support future business features such as cash register tracking, stock movements, and sales reports.

---

## Features

- Product Management.
- Category Management.
- Sales registration.
- Inventory movements.
- Cash register control.
- Product stock tracking.
- Authentication and user roles.

---
## Tech Stack

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
- Lombok
- Maven
